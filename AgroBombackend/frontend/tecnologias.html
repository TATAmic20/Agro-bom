Como o front-end foi desenvolvido

Esta tela apresenta, de forma simples, as tecnologias utilizadas na construção da interface visual do AGROBOM.


Resumo da minha parte

Minha responsabilidade foi desenvolver as telas do sistema, deixando a navegação organizada e criando uma interface fácil de entender para o usuário.



HTML

Usado para montar a estrutura das páginas, como menus, títulos, tabelas, botões e formulários.

CSS

Usado para definir as cores, espaçamentos, tamanhos, layout, menu lateral e aparência do sistema.

JavaScript

Usado para controlar interações da interface, como abertura de modais, cadastro, edição e exclusão.

Font Awesome

Usado para adicionar ícones ao menu lateral, botões e outras partes visuais do sistema.

Simulação de Dados

Durante o desenvolvimento do front-end, alguns dados foram simulados para testar as telas antes da integração.

Organização

O projeto foi dividido em páginas separadas, facilitando a navegação e a manutenção das telas.

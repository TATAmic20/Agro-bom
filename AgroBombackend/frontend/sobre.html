AGROBOM 🌱

Sistema acadêmico desenvolvido para auxiliar na gestão de empresas do setor agrícola, oferecendo uma interface simples, organizada e intuitiva.


Objetivo do Projeto

O AGROBOM foi desenvolvido como projeto acadêmico com o objetivo de demonstrar a aplicação de conceitos de gestão, organização de dados e desenvolvimento web em um ambiente voltado para o setor agrícola.



Problema que o Sistema Resolve

Muitas empresas precisam controlar clientes, produtos, pedidos e estoque simultaneamente. O sistema busca reunir essas informações em um único ambiente para facilitar a administração do negócio.



Principais Funcionalidades

  • Cadastro de clientes
  • Gerenciamento de produtos
  • Controle de pedidos
  • Monitoramento de estoque
  • Visualização de relatórios
  • Painel administrativo


Minha Participação no Projeto

Fiquei responsável pelo desenvolvimento do front-end do sistema, criando as telas, menus, formulários e elementos visuais responsáveis pela interação do usuário com a aplicação.



Tecnologias Utilizadas

O desenvolvimento da interface foi realizado utilizando HTML para estruturação das páginas, CSS para estilização visual e JavaScript para as funcionalidades e interações presentes no sistema.
